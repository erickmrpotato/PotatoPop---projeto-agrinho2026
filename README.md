PotatoPOP!
O PotatoPOP! é o cantinho mais crocante da internet! Um portal de notícias e opiniões totalmente focado no ecossistema geek e na cultura pop, cobrindo o multiverso do entretenimento: desde os maiores lançamentos de games até filmes que acabaram de sair no cinema.
Concurso Agrinho 2026
Este projeto foi desenvolvido especialmente como parte das atividades e submissões para o Concurso Agrinho 2026.

Tecnologias Utilizadas
O site foi construído utilizando as tecnologias fundamentais do desenvolvimento web para garantir leveza, velocidade e compatibilidade total:

HTML5: Estruturação semântica de todas as páginas e seções.
CSS3: Estilização moderna, sistema de cores focado no modo escuro (Dark Mode), design responsivo e efeitos de transição/hover de alta fidelidade.
JavaScript (Vanilla): Implementação de interações leves e controle de comportamento de tela.
Estrutura do Projeto
O repositório está organizado de forma simples e direta na raiz para facilitar a hospedagem estática:

index.html — Página principal de boas-vindas e introdução ao portal.
noticias.html — Aba funcional com os cartões de notícias organizados de forma responsiva.
sobre.html — Página institucional detalhando a missão e o propósito do PotatoPOP!.
styles.css — Arquivo único de estilização contendo todas as regras visuais, paleta de cores e comportamento responsivo (adaptação para computadores e celulares).
Como Visualizar o Projeto
Como o site é composto por páginas estáticas, você não precisa instalar nenhuma dependência:

Faça o download ou clone este repositório.
Navegue até a pasta do projeto.
Dê um duplo clique no arquivo index.html para abrir diretamente no seu navegador de preferência.
Alternativamente, o projeto está configurado e publicado online através do GitHub Pages. O link para acesso direto pode ser encontrado na seção About deste repositório.
